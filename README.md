# Exchange Rate Dashboard
## Projektziel
Entwicklung einer Fullstack-Webanwendung mit React, Node.js und TypeScript zur Darstellung aktueller und historischer Wechselkurse.
Das Backend stellt eine REST-API bereit und verarbeitet Daten einer externen Wechselkurs-API. Zusätzlich sollen Favoriten gespeichert und verwaltet werden.

## Technologien

### Frontend
- React
- TypeScript

### Backend
- Node.js
- Express

### Datenbank
- MongoDB

## Geplante Funktionen
- Aktuelle Wechselkurse anzeigen
- Historische Kursverläufe visualisieren
- Favoriten speichern
- Favoriten verwalten
- Responsive Benutzeroberfläche
